## Features

- **Real-time preview:** See changes instantly as you edit
- **ATS optimization score:** 0-100% feedback on ATS-friendliness
- **Dual editor modes:** Form editor and JSON editor
- **Auto-save:** Changes persist to your browser automatically
- **Multiple export formats:** PDF, HTML, JSON, and plain text
- **Responsive design:** Works on desktop, tablet, and mobile
- **Data management:** Reset to template or clear all data with confirmation

---

## Quick Start

1. Add your Personal Information (name, email, phone, location) and a professional summary
2. Add Work Experience with bullet-point achievements
3. Add Education (degrees, schools, dates, and optional score/CGPA)
4. Add Skills (grouped by category)
5. Add Projects and Certifications (optional)
6. Check your ATS Score and adjust content for optimization
7. Export your resume (PDF/HTML/JSON/Text) and apply to jobs

## Resume Structure

- Personal Information
- Work Experience
- Education (including score/CGPA)
- Projects
- Certifications & Achievements
- Skills

ATS-friendly by design: single-column layout, no images or complex formatting, semantic HTML structure for parsing.

---

## Exports

- PDF: Print-ready, ATS-friendly text-based PDF
- HTML: Standalone, shareable HTML file
- JSON: Data export for backup or editing
- Text: Copy-friendly plain text for quick applications

---

## Data & Privacy

- 100% local storage: data stays on your device
- No cloud sync or tracking
- No data is sent to servers
